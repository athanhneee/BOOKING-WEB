# BOOKING-WEB
Trang web đặt phòng lưu trú Vũng Tàu là nền tảng đặt phòng biệt thự và khu nghỉ dưỡng lấy cảm hứng từ Airbnb và Booking.com. Dự án được xây dựng với React, TypeScript, TailwindCSS cho frontend và Node.js, Express.js cho backend. Cho phép người dùng khám phá điểm đến, tìm kiếm, xem chi tiết và đặt phòng với giao diện trẻ trung, trải nghiệm mượt mà
